Hitta

Hitta is a library for querying data in the browser using MongoDB-like queries. The following example will select all elements in the array 'users' with a name found in the array defined after $in. 

    let queried = query(users)({
        name: {
            $in: ['David', 'Daniel', 'Clark']
        }
    });

Reference

$gt

Select items where the value is greater than the query value

    let queried = query(users)({
        age: {
            $gt: 20
        }
    });
$gte

Select items where the value is greater than or equal to the query value

    let queried = query(users)({
        age: {
            $gt: 20
        }
    });
